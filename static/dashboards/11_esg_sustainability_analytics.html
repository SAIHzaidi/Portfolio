Python — Emissions Forecast to 2030
Linear regression + policy scenario modelling · scikit-learn · pandas
scikit-learn · pandas
# Emissions forecast model — 3 scenarios to 2030 # Scope: Scope 1+2 (verified) · FY2019–FY2024 actuals import pandas as pd import numpy as np from sklearn.linear_model import LinearRegression from sklearn.preprocessing import PolynomialFeatures # ── Load and prepare actuals ────────────────────────────────────── df = pd.read_csv('ghg_actuals_2019_2024.csv') df['year_idx'] = df['year'] - 2019 X = df[['year_idx']].values y = df['total_tco2e'].values # ── Fit polynomial regression (degree 2) ───────────────────────── poly = PolynomialFeatures(degree=2, include_bias=False) X_poly = poly.fit_transform(X) model = LinearRegression().fit(X_poly, y) # ── Forecast years 2025–2030 ───────────────────────────────────── future_years = np.arange(2025, 2031) X_future = poly.transform((future_years - 2019).reshape(-1, 1)) baseline_forecast = model.predict(X_future) # ── Policy scenarios ───────────────────────────────────────────── scenarios = { 'BAU': baseline_forecast, 'Moderate action': baseline_forecast * np.linspace(1.0, 0.82, 6), 'Accelerated': baseline_forecast * np.linspace(1.0, 0.60, 6), } # ── Check against Net Zero pathway ─────────────────────────────── target_2030 = 41490 * (1 - 0.40) # −40% vs 2019 = 24,894 tCO₂e print(f"2030 target: {target_2030:,.0f} tCO₂e") for name, values in scenarios.items(): gap = values[-1] - target_2030 status = '✓ On track' if gap <= 0 else f'⚠ Gap: {gap:,.0f} tCO₂e' print(f"{name}: {values[-1]:,.0f} tCO₂e → {status}")

Accelerated scenario meets 2030 target

Accelerated scenario closes the gap — requires 3 key interventions

1. Full renewable electricity procurement across all sites by FY26 (est. −6,200 tCO₂e). 2. Fleet electrification: 80% EV by FY27 (est. −2,800 tCO₂e). 3. Supplier engagement programme driving Scope 3.1 reduction of 15% (est. −2,520 tCO₂e). Combined: −11,520 tCO₂e, closing the projected 8.8pp gap.
