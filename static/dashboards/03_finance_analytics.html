Technology overspend is the critical finding

Technology costs exceeded budget by £640K (−20%). Deep-dive analysis revealed: 34 SaaS subscriptions with <5 active users (£290K), unoptimised cloud infrastructure (£180K estimated waste via rightsizing analysis), and 2 duplicate vendor contracts (£88K). Total recoverable: £558K annually.

04 · The SQL — Variance Decomposition
-- Budget vs Actual variance decomposition -- Splits total variance into Volume, Price, Mix effects -- Used for monthly CFO board pack automation WITH actuals AS ( SELECT department, cost_category, SUM(amount) AS actual_spend, COUNT(invoice_id) AS invoice_count FROM finance.gl_transactions WHERE fiscal_year = 2024 GROUP BY 1,2 ), budget AS ( SELECT department, cost_category, SUM(budget_amount) AS budget_spend FROM finance.annual_budget WHERE fiscal_year = 2024 GROUP BY 1,2 ) SELECT a.department, a.cost_category, a.actual_spend, b.budget_spend, a.actual_spend - b.budget_spend AS variance_abs, ROUND( (a.actual_spend - b.budget_spend) / b.budget_spend * 100, 1 ) AS variance_pct, CASE WHEN (a.actual_spend - b.budget_spend) / b.budget_spend > 0.10 THEN 'MATERIAL OVERSPEND' WHEN (a.actual_spend - b.budget_spend) / b.budget_spend < -0.10 THEN 'MATERIAL UNDERSPEND' ELSE 'WITHIN TOLERANCE' END AS flag FROM actuals a JOIN budget b USING(department, cost_category) ORDER BY ABS(variance_abs) DESC

Automated the monthly board pack

This query, wrapped in a Python script, auto-populates a Power BI dataset and Excel template on the 3rd working day of each month. Reduced Finance team's reporting time from 3 days to 4 hours per month.
